# New Team Member Onboarding Guide

## Prerequisites

Install:

* Git
* Docker
* VS Code
* Jenkins
* GitHub Account

## Setup Steps

1. Clone repository.
2. Create a feature branch.
3. Make changes.
4. Commit code.
5. Push changes.
6. Create pull request.

## CI/CD Workflow

Developer → GitHub → Pipeline → Build → Test → Deploy

## Best Practices

* Commit frequently.
* Write meaningful commit messages.
* Review pipeline logs.
* Follow security guidelines.
